Matrix Spark

The [http://en.wikipedia.org/wiki/Spark_(mathematics), spark] of a matrix---useful in [http://en.wikipedia.org/wiki/Compressed_sensing, compressive sensing]---is the smallest number of [http://en.wikipedia.org/wiki/Linear_independence,linearly dependent] columns. For example, if some linear combination of three columns sums to zero, but no combination of two columns sums to zero, the spark is three. Calculating the spark involves checking the [http://en.wikipedia.org/wiki/Linear_independence, dependence] of combinations of columns. This can be expensive, or practically impossible. Nevertheless, it is instructive to compute easy examples.
The code below checks the [http://en.wikipedia.org/wiki/Rank_(linear_algebra),rank] for combinations of $k$ columns, for increasing $k$. If a combination is found with rank less than $k$, then $k$ is the spark. E.g., let $$ A=\begin{bmatrix} 1&0&1&1\\0&1&1&-1 \end{bmatrix} $$ It is easy to check that no two columns of $A$ are linearly dependent. Since we can find three columns that sum to zero the spark is three (the maximum).

Code

The CoffeeScript can be edited and computes in the browser. The Octave/Matlab code can be executed locally after cloning / downloading [https://github.com/stemblab/matrix-spark, this repository] (see the m/ directory).
E.g., add the first two columns and subtract the third.
The maximum possible rank is one more than the number of rows. In the example the maximum spark is three because it is impossible to find three independent columns (of length two).
